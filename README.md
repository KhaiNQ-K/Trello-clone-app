# Trello app clone with ReactJS + Hooks + Redux

## Requirements

- **nodejs >= v14.7.**
- **npm >= v6.14.7**
- **yarn >= v1.19.1**
  Clone project and run test.

```

$ git clone https://github.com/KhaiNQ-K/trello-clone-app.git
$ cd <project_dir>
$ yarn install
$ yarn start
```
